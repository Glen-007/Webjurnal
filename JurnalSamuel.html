Judul Jurnal: Penerapan dan Pengaruh Struktur Data dalam Pengembangan Perangkat Lunak

Penulis: Samuel Glen Jhonniari Marpaung

Tanggal: 28 Mei 2025

Abstrak

Struktur data merupakan komponen penting dalam ilmu komputer dan rekayasa perangkat lunak. Penelitian ini membahas jenis-jenis struktur data dasar seperti array, linked list, stack, queue, tree, dan graph serta aplikasinya dalam berbagai aspek pengembangan perangkat lunak. Tujuan dari jurnal ini adalah untuk memahami bagaimana struktur data mempengaruhi efisiensi dan performa algoritma serta memberikan dasar pemahaman bagi pengembang dalam memilih struktur data yang tepat untuk menyelesaikan masalah tertentu.

Pendahuluan

Struktur data adalah cara menyimpan dan mengatur data agar dapat digunakan secara efisien. Dalam dunia pemrograman, pemilihan struktur data yang tepat sangat penting untuk mengoptimalkan performa aplikasi. Penggunaan struktur data yang tidak sesuai dapat menyebabkan program menjadi lambat atau tidak efisien.

Jenis-Jenis Struktur Data

Array: Array adalah struktur data yang menyimpan elemen dalam satu blok memori yang berurutan. Array cocok digunakan ketika jumlah elemen diketahui dan akses indeks cepat diperlukan. Linked List: Linked list terdiri dari elemen-elemen yang saling terhubung dengan pointer. Kelebihannya adalah mudah menambah dan menghapus elemen, tetapi akses acak menjadi lebih lambat. Stack: Stack menggunakan prinsip LIFO (Last In First Out). Banyak digunakan dalam pengolahan ekspresi, undo-redo, dan pemanggilan fungsi rekursif. Queue: Queue bekerja dengan prinsip FIFO (First In First Out). Cocok digunakan untuk antrian data, seperti pada sistem cetak atau penjadwalan proses. Tree: Tree adalah struktur data hierarkis. Salah satu jenis yang populer adalah binary search tree yang efisien dalam pencarian, penambahan, dan penghapusan data. Graph: Graph digunakan untuk merepresentasikan hubungan antar objek, seperti jaringan komputer, peta rute, dan algoritma pencarian jalur.

Penerapan dalam Dunia Nyata

- Array digunakan dalam pengolahan gambar atau matriks numerik.

- Stack dan Queue digunakan dalam manajemen proses, sistem navigasi, dan antrian sistem.

- Tree digunakan dalam database (seperti B-Tree) dan sistem file.

- Graph digunakan dalam media sosial, pencarian rute, dan jaringan komunikasi.

Kesimpulan

Struktur data merupakan fondasi penting dalam pengembangan perangkat lunak. Pemilihan struktur data yang tepat tidak hanya membuat program lebih efisien tetapi juga lebih mudah dikelola dan dikembangkan. Dengan memahami karakteristik masing-masing struktur data, pengembang dapat merancang solusi perangkat lunak yang optimal.
